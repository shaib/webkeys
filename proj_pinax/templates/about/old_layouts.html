{% trans "Where did the old layouts go?" %}

This site had an initial version where some users created layouts; what happened to these layouts?

The initial version of this site had no concept of ownership, and all layouts belonged to all users; so anyone could edit any layout. This is no longer the case -- with this version and going forward, users can only edit their own layouts.

To preserve the old work, all old layouts have been attached to a "fake" user named "old_times". If you had a layout in the old site, you can still find it in old_times' profile. If you want to modify it, just clone it and continue working on your own copy. You will need to register for this, of course.
