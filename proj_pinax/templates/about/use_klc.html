{% trans "Using KLC files on Windows" %}

To use the KLC files produced by this site, you need the microsoft MSKLC utility.

Download and install MSKLC

MSKLC – The Microsoft Keyboard Layout Creator – is freely available from Microsoft. It is described here, and can be downloaded from here.

After you download and install it, it will be the default application to handle .klc files, so files you generate on this site will be automatically opened in MSKLC. To install the layout, you need to export an installer from within MSKLC, and then install it. You can, of course, also distribute this installer to other users, for them to use your layout.
